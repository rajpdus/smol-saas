# Tests for smol-saas

This directory contains tests for the smol-saas CLI tool.

## Running Tests

Run all tests with pytest:

```bash
python -m pytest
```

Run a specific test:

```bash
python -m pytest tests/test_basic.py
```

## Test Structure

- `test_basic.py`: Basic tests for the CLI commands and functionality
- Additional test files will be added as the project grows
